# REALIA 3D Open World (A) - Complete Demo

「ブラウザで動く3Dオープンワールド + 服を着た3Dキャラ + 三人称視点 + 入場ゲート + AI接客（簡易）」のMVPです。

## ファイル（4つだけ）
- index.html
- styles.css
- app.js
- README.md

## 実装済み（重要）
- **服を着た3Dキャラ**（GLB）を読み込み
- **三人称視点（TPS）カメラ**で追従
- **歩行アニメーション**（Idle / Walk）
- **スマホ：左下スティックで移動**、PC：WASD/矢印
- **複数エリア**（ゲート別で入場）
- **AIポイント**に近づくと「話す」が有効
- **右上ミニマップ**（あなた/ゲート/AI）

## 外部読み込みについて
- Three.js は CDN から読み込み
- 3Dキャラは threejs.org の公開GLBを読み込み
  - 最初だけ数秒かかる場合があります（通信）

## Render（静的サイト）設定
- Build Command：空（不要）
- Publish Directory：`.`（ドット）

## 次の段階（あなたの目標画像に近づける）
- 地面や道路を高品質テクスチャに
- 本格的な街のプロップ（看板、街灯、車、歩行者）
