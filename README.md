# REALIA Simple Demo (No Build)

これは「雰囲気チェック用」の超簡単版です。ビルド不要で、そのまま静的サイトとして公開できます。

## ファイル
- index.html
- styles.css
- app.js

## Render（静的サイト）で公開する
1) Renderで「Static Site（静的サイト）」を作成
2) このリポジトリを選ぶ
3) 設定
   - Build Command: （空欄でOK / もし必須なら `echo ok`）
   - Publish Directory: `.`
4) Deploy

## GitHub Pagesで公開する（任意）
- Settings → Pages → Branch: main / root → Save
