# REALIA ブラウザ オープンワールド（完全デモ / A）

これは **ビルド不要（静的サイト）** で動く、オープンワールド風 + AI接客（ダミー）デモです。

- ファイル：`index.html` / `app.js` / `styles.css` / `README.md`
- 外部ライブラリなし（スマホでも動きやすい）
- Render / GitHub Pages どちらでもOK

## 操作

- PC：WASD / 矢印キーで移動
- スマホ：左下のスティックで移動（キャンバスをタップでも反応）
- ゲート（青い円）に近づくと「このエリアに入る」が有効になります
- NPC（紫）に近づくと「話す」が有効になります

## エリア（入口別）

ワールド内に入口（ゲート）を **3つ** 搭載しています。

- Cyber Roppongi (A)：夜 / ネオン / 霧
- Green Avenue：昼 / 緑 / 未来街路灯
- Tower District：夕方 / 高層ビル街

## Render（静的サイト）設定

- Build Command：空欄
- Publish Directory：`.`
