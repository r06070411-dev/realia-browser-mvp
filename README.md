# REALIA Browser Open World (Complete Demo)

ビルド不要の静的サイトです（Render Static Site / GitHub Pages でそのまま動きます）。

## ファイル
- index.html
- app.js
- styles.css
- README.md

## 操作
- PC: WASD / 矢印キー
- スマホ: 左下のスティック

## 入口
- 入口A: サイバー（ネオン/霧/雨）
- 入口B: 高級近未来（洗練）

## エリア
入口ごとに複数エリア（ゲート）があります。ゲートに近づくと入れます。

※AI応答はダミーです。後で本物のAI APIに差し替えできます。
