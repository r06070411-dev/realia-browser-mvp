# REALIA ブラウザ・オープンワールド（完全デモ A）

これは「ブラウザ上で動くオープンワールド + AI接客（簡易）」の最小MVPです。

## ファイル構成（この4つだけ）
- index.html
- styles.css
- app.js
- README.md

## できること
- スマホ：左下スティックで移動
- 右上ミニマップ表示
- ワールド内の「ゲート（緑）」に近づくと「このエリアに入る」ボタンが有効
- 複数エリア（Fashion / Reform / Food）をゲート別で搭載
- AIポイント（紫）に近づくと「話す」が有効＆タップでも話せる
- チャットは現状ダミー応答（後で本物のAIに差し替え可能）

## Render（静的サイト）設定
- Build Command：空（または ` `）
- Publish Directory：`.`（ドット）
- Auto Deploy：ON（推奨）

## 注意
- GitHubの表示がカタカナになるのは「ファイル名」ではなく、iPhone側の表示/翻訳UIの影響で起きることがあります。
- 実際に Render は `index.html / app.js / styles.css` を読みます。ファイル名は必ずローマ字のままにしてください。
